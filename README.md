# yii2-helpdesk

Install
---------------------------------

Run

```
php composer require pantera-digital/yii2-helpdesk "*"
```

Or add to composer.json

```
"pantera-digital/yii2-helpdesk": "*",
```

and execute:

```
php composer update
```

Run migrations:

```
php yii migrate --migrationPath=vendor/pantera-digital/yii2-helpdesk/migrations
```
